# SOC Attack Detector

A Python-based Security Operations Center (SOC) tool that analyzes authentication logs to detect brute-force login attempts.

## Features

- Detects multiple failed login attempts
- Extracts suspicious IP addresses
- Generates SOC alert reports

## Use Case

SOC analysts use log analysis to identify brute-force attacks, unauthorized access attempts, and suspicious activity in authentication logs.

## Technologies

- Python
- Regular Expressions
- Log Analysis
